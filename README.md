# 📈 Trading Bot (Python)

A simple algorithmic trading bot built using Python that demonstrates automated trading logic for MARKET and LIMIT orders. This project is designed for learning core concepts of algorithmic trading, order execution, and logging systems.

---

## 🚀 Features

- 📊 Simulates MARKET orders
- 📉 Simulates LIMIT orders
- 🧾 Logs all trades into log files
- ⚙️ Simple trading logic for educational use
- 🧪 Helps understand basic algorithmic trading structure

---
